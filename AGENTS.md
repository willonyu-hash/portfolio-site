# AGENTS.md

## Project Goal
This repository is a premium personal portfolio microsite for a fashion trend designer in the sports category. The site is designed for resume QR code usage, job interviews, and portfolio presentation.

## Product Priorities
1. Mobile-first experience
2. Premium minimal visual quality
3. Editable content structure
4. Clean code and maintainable architecture
5. Fast deployment to Vercel

## Design Direction
- luxury minimal
- editorial
- fashion-forward
- refined, spacious, modern
- calm neutral palette
- avoid generic developer portfolio aesthetics
- avoid over-animation
- avoid cheap gradient-heavy UI

## Technical Rules
- Use Next.js and Tailwind CSS
- Keep content and UI separated
- Prefer JSON or Markdown driven content
- Build reusable components
- Optimize for image-led storytelling
- Keep the codebase easy for future content updates

## Working Rules
- Before coding, briefly state the plan
- Make focused and minimal changes
- After each task, summarize:
  - what changed
  - which files changed
  - why
  - what to do next
- If there are actual file changes, run git add and git commit
- Use clear commit messages with prefixes like feat, fix, refactor, docs, chore

## Audience
Interviewers, hiring managers, and team leads in fashion, trend, creative, and innovation-related roles.

## Content Tone
Professional, concise, elevated, credible, modern.
