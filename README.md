# Fashion Trend Portfolio Microsite

Premium, mobile-first personal portfolio microsite built with Next.js and Tailwind CSS.

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Content Editing

Most future updates only need these files:

- `src/content/profile.ts`
- `src/content/projects.ts`
- `src/content/reports.ts`
- `src/content/ai-cases.ts`
- `src/content/contact.ts`

Detailed editing notes live in `docs/content-maintenance.md`.
